# CO2 Adsorption Experiment Analyzer

## Overview
This project is a simple scientific tool designed to analyze CO2 adsorption experiment data. It helps researchers and students automatically calculate adsorption performance and visualize results.

## Problem
In many laboratories, adsorption data is processed manually using spreadsheets. This process is slow, repetitive, and prone to errors.

## Solution
This tool automates the analysis by:
- Calculating adsorption capacity (qₜ)
- Calculating removal efficiency (%)
- Generating plots (concentration vs time, adsorption vs time)
- Comparing different adsorbent materials

## Features
- Upload CSV or Excel data
- Automatic calculations
- Data visualization (graphs)
- Performance comparison between samples
- Export results

## How to Use
1. Run the app:
   streamlit run app.py

2. Upload a dataset (sample_data.csv)

3. View:
   - Tables
   - Graphs
   - Summary results

## Formula Used
qₜ = (C0 - Ct) × V / m

## Example Insight
The tool shows that thiourea-doped carbon performs better than standard activated carbons in CO2 adsorption.
