STEP-BY-STEP SUBMISSION GUIDE

1. Extract the ZIP file.
2. Create a GitHub repository named `co2-adsorption-experiment-analyzer`.
3. Upload:
   - app.py
   - requirements.txt
   - sample_data.csv
   - README.md
   - submission_explanation.txt
4. Run locally:
   pip install -r requirements.txt
   streamlit run app.py
5. Upload sample_data.csv in the app and confirm tables and plots appear.
6. Record a demo under 2 minutes.
7. Submit the GitHub link and explanation in the Google Form.
